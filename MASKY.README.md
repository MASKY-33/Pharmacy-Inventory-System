# Pharmacy Inventory System
This system allows the user to search for a medicine and see in which cabinet it is stored.
The program keeps running until the user types "stop", making it useful for quick night‑shift lookups.

## Features
- Stores medicines and their cabinet locations in a dictionary
- Lets the user search any medicine by name
- Checks if the medicine exists in stock
- Shows the correct cabinet when found
- Gives a clear message if the medicine is not in stock
- Runs in a continuous loop until the user types "stop"

## How to use
- Run the system and type the name of any medicine.
- The program will show the cabinet where it is stored.
- Type "stop" to exit the system.
